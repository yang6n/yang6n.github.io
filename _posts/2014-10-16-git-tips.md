####Revert "git commit" after "git push"
```bash
git reset --hard HEAD~1
git push -f
```

####Remove tag from remote
```bash
git tag -d 12345
git push origin :refs/tags/12345
```

###Remove remote branch
```bash
git push origin --delete branch_name
```
